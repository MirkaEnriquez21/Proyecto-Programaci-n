\# Proyecto de Programación



\## Descripción



En esta presentación se expondrá la implementación de tres algoritmos para resolver diferentes problemas matemáticos mediante programación, con el objetivo de aplicar los conocimientos adquiridos en programación mediante la implementación de algoritmos en lenguaje C++ para resolver diferentes problemas.



\## Funcionalidades



\- Entrenar una red neuronal para predecir el valor de y teniendo una entrada x.

\- Resolver ecuaciones complejas.

\- Resolver sistemas de ecuaciones lineales.



\## Cómo ejecutar el programa



1\. Abrir el proyecto en Dev-C++.

2\. Compilar el programa.

3\. Ejecutar el archivo.

4\. Seleccionar que ejercicio desea ejecutar.
